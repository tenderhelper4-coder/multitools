Frequently Asked Questions (FAQ)

1. What is EMD?
Earnest Money Deposit (EMD) is a security amount submitted along with the tender bid to ensure seriousness of the bidder.

2. What is Security Deposit?
Security Deposit or Performance Security is deposited after tender award to ensure proper execution of work.

3. How is EMD calculated?
EMD = (Tender Value × EMD Percentage) ÷ 100

4. Can I use this for government tenders?
Yes, this tool works for PWD, CPWD, Railways, Municipal, and other government tenders.
